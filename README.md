Project Overview
This project demonstrates an end-to-end data pipeline, transforming raw YouTube data into an interactive business dashboard. The workflow covers the entire data lifecycle: Extraction, Cleaning (Python), Analysis (SQL), and Visualization (Power BI).

🛠 Tools & Technologies
Data Processing: Python (Pandas)

Database & Analysis: SQL (PostgreSQL/MySQL/SQL Server)

Visualization: Power BI

Presentation: Gamma AI & MS Word

🚀 Project Workflow
Data Cleaning & Transformation: * Utilized Pandas to handle unstructured raw data from YouTube.

Performed data type conversions, handled missing values, and standardized schemas for database ingestion.

SQL Analysis: * Migrated cleaned data into a relational database.

Executed complex SQL queries to perform deep-dive analysis on engagement metrics and audience trends.

Power BI Dashboarding: * Connected the SQL database to Power BI.

Built an interactive dashboard featuring key performance indicators (KPIs), trend analysis, and category-based filtering.

Reporting: * Compiled insights into a structured professional report.

Generated an AI-powered presentation using Gamma to summarize the project's technical architecture and business findings.

📉 Dashboard Highlights
Engagement Tracking: Visualized Likes, Shares, and Comment ratios.

Growth Trends: Identified peak performance periods through time-series analysis.

Category Analysis: Comparative view of performance across different content genres.

⚙️ How to Run
Step 1: Run the Python scripts included in this repository to process the raw data.

Step 2: Import the resulting datasets into your SQL environment and execute the provided analysis queries.

Step 3: Open the Power BI file to explore the interactive visualizations and final presentation.
